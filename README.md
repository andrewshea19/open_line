# Open.Line

See when friends are free to talk. Share your availability status in real-time.

## Features

- Set your status to Available, Unavailable, or No Status
- See which friends are free to call at a glance
- Schedule recurring availability (Mon-Fri, weekends, custom)
- One-time events for specific dates
- Home screen and lock screen widgets
- One-tap calling when friends are available
- Secure sync via iCloud

## Requirements

- iOS 17.0+
- Xcode 15+
- Apple Developer account (for CloudKit)

## Setup

1. Clone the repository
2. Open `OpenLine.xcodeproj` in Xcode
3. Update the bundle identifier to your own
4. Configure CloudKit container in Apple Developer Portal
5. Build and run

## Privacy

Open.Line uses iCloud to sync your availability status with friends. Your data is only shared with friends you explicitly approve. See our [Privacy Policy](docs/PRIVACY.md) for details.
